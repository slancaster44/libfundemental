Detailed Description

Memory management arena implementation. Enables grouping allocations together, so they may be free'd together

◆ ARENA_BLOCK_SIZE

#define ARENA_BLOCK_SIZE   4096

Size of arena block when allocated. When the block is full a new block will be allocated using 'malloc().' When an allocation is larger than ARENA_BLOCK_SIZE, a special block, large enough for the allocation, will be malloc'd

The 'none' arena. This is not a real arena. Allocations against this arena will simply call malloc. It is up to the user to free the returned memory themselves.
