Create a list node of a given length, and copy the data into it. The next and previous elements will be initialized to the same node

Parameters
arenaThe arena to allocate against. ARENA_NONE will simply use malloc(). Calling free() on a ListNode_t is only valid if it was allocated against ARENA_NONE
dataThe data to copy into the list node. If it is NULL, the node's data section will be left uninitialized
lengthThe length of the data section. The total size of the node can be given by the LIST_NODE_SIZE macro
