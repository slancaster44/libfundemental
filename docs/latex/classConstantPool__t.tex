\doxysubsection{Detailed Description}
A pool of constant objects ~\newline
. 

Interning a value against a pool will either\+: ~\newline
 A) Add a new constant object to the pool and return a pointer to that constant object ~\newline
 B) If the an identical constant object already exists in the pool, return a pointer to that constant object
