\doxysubsection{Detailed Description}
The iterator object. Used for looping over a collection objects (eg \mbox{\hyperlink{classList__t}{List\+\_\+t}}, \mbox{\hyperlink{classBuffer__t}{Buffer\+\_\+t}}). 

It is strongly recommended that data structions not be modified while they are being iterated over.
